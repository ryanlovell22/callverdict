Where do I find my Twilio credentials?
  1. Log in to your Twilio Console
  2. Your Account SID and Auth Token are on the main dashboard
  3. Click "Show" next to the Auth Token to reveal it, then copy both values here

Where do I find my CallRail API key?
  1. Log in to CallRail
  2. Go to Settings (gear icon) → API
  3. Under API Keys, copy your existing key or create a new one
